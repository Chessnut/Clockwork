About
Clockwork
Clockwork is an open source roleplaying systems framework. Feel free to use it as you wish, but we at Cloud Sixteen only ask you respect us and our efforts and help us counter piracy of our schemas.

A list of Cloud Sixteen schemas can be found or bought here, please do not pirate these schemas; instead become a fully licensed Clockwork customer and receive free lifetime updates of both Clockwork and the schema.
Cloud Sixteen
Cloud Sixteen is a small indie games company focused on creating fun games. Although they have moved on to developing their own 2D and 3D engine, they still play a big part in developing one of Garry's Mod's widely used roleplaying frameworks.

Cloud Sixteen is also a gaming community, and anybody is welcome at the forums if they have any questions, suggestions, bug reports or just want to hang out.
Ideal-Hosting
Ideal-Hosting manage the licensing and distribution of both Cloud Sixteen schemas and free public development keys. Cloud Sixteen and Ideal-Hosting have been business partners since 2010.
Install
Clockwork
  • Extract the files in this archive to just outside of your garrysmod/ directory and overwrite any existing files.
  • Edit garrysmod/gamemodes/Clockwork/mysql.cfg with your own MySQL database information and save the file.
  • Enter your Steam ID into garrysmod/gamemodes/Clockwork/owner.cfg to become a super admin.
    • Schema
  • Place the <schema> directory into garrysmod/gamemodes/ and overwrite any existing files.
  • Place any content such as materials, models, scripts and sounds into the garrysmod/ directory and overwrite any existing files.
  • Enter your serial key into garrysmod/gamemodes/<schema>/serial.cfg and save the file.
  • Set sv_defaultgamemode to whatever the folder name of the <schema> is, e.g: sv_defaultgamemode cwHL2RP.
  • You should be all set from there. Load up your server and enjoy.
    • Issues
    If you have any issues with Clockwork or you want to report a bug or even give a suggestion, the following are the best places to try:
  • The Cloud Sixteen project tracker.
  • The Cloud Sixteen community forums.
  • The Ideal-Hosting billing and support website.
    • Linux
  • You need to install libboost_thread-mt.so.1.38.0. or apt-get install libboost-all-dev from your package manager. If you cannot find them, you may be able to use the files in libboost/ of this directory, they belong in the LD_LIBRARY_PATH ex /usr/lib.
  • You need to install libmysqlclient or libmysqlclient16 or newer via your package manager.
  • When dealing with file directories, names or paths, do not forget Linux is case-sensitive!
    • Windows
  • Make sure you have installed the VC++ Redist 2005/2008 libraries.
